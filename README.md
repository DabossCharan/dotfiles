## Installation

### 1. Install Arch Linux.
Pretty Important I Think.
### 2. Clone Repository.
```
git clone https://github.com/DabossCharan/dotfiles.git
```
### 2. Change Directory.
```
cd dotfiles
```
### 3. Start Installation
```
sh install.sh
```
